# Relaciones Entre Tablas 
En las bases de datos relacionales las tablas no están aisladas, sino que se conectan entre sí para organizar la informción. Existen orinicipalmente dos tipos fundamentales de relacionar tablas:
## Foreign Key
Se utiliza cuando existe una relación 1:N dada cuando un registro de una tabla está relacionado con varios registros de otra pero cada registro de la segunda pertenece a solo uno de la primera.
## Tabla Intermedia
Se utiliza cun¿ando existe una relación N:M dada cuando varios registros de una tabla pueden relacionarse con varios de otra creando así una tabla intermedia con las Foreign Keys de ambas tablas.
