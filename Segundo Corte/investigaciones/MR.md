# MR
El modelo relacional es un esquema utilizado oara organizar los datos en una base de datos usando tablas. En este modelo los datos se estructuran en tablas compuestas por filas y columnas.
Cabe aclarar que el modelo relacional transforma los diseños de los diagramas anteriores vistos en clase (UML, MER) en estructuras implementables en sistemas de bases de datos relacionadas como MySQL.
## Filas
Cada fila representa un registro único insertado en la base de datos.
## Columnas
Cada columna corresponde a un atributo o característica de un dato.
