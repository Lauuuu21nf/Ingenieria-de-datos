Se crea con el USE
Todas las colecciones van el minuscula

cuando se inserta solo un documento se hace 
db.asistencia.insertOne({comentariosCapacitacion: "Muy bien", asistio: 1})


cuando se insertan varios documentos, se hace como si fuera un array, entonces queda 
db.trabajadores.insertMany([{nombre: "Tatiana", empresaDondeTrabaja: "UROSARIO", documentoTrabajador: 1024435808},{nombre: "José", empresaDondeTrabaja: "SENA", documentoTrabajador: 254684154}])


para consultar que los elementos se hayan creado
db.trabajadores.find()

lo mismo de arriba pero para que se vea con una mejor estructura
db.trabajadores.find().pretty()

si solo queremos buscar UN documento queda
db.usuario.findOne({'nombreUsuario': "Xavier Enrique Naar Amaris"})

buscar con OR
db.usuario.find({$or: [{rolUsuario: "admin"}, {estadoUsuario: "activo"}]})

buscar con AND
db.usuario.find({$and: [{rolUsuario: "admin"}, {estadoUsuario: "activo"}]})

siempre que se pone $ significa que va a buscar un valor

buscar diferentes a lo que se especifica
db.trabajadores.find({empresaDondeTrabaja: {$ne: "TGI"}})

consulta implícita es un AND, entonces solo se separa por una coma las propiedades y valores
{$gte: 18, $lte: 30}

db.collection.find({country: {$in["EEUU", "Col"]}})

buscar si existe un documento con un valor, o es true o es false (en el ejemplo, si tiene telefono)
db.collection.find({telefono: {$exists: true}})


si se van a hacer estas mismas consultas en compass, solo se copia lo que esta adentro de los corchetes

cambiar todos los total
db.productos.updateMany({}, [{$set: {total: {$multiply: ["$precioUnitario", "$cantidad"]}}}])
